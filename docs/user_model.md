# Model użytkownika

## Cel

Model użytkownika przechowuje informacje potrzebne do personalizacji działania Ogarniacza Życia.

Nie jest to tylko lista ustawień. Jest to dynamiczny profil, który rozwija się razem z użytkownikiem.

---

# Główne obszary profilu

## 1. Podstawowe informacje

Przechowywane informacje:

- imię,
- preferowany sposób komunikacji,
- dostępny czas,
- typowy harmonogram dnia.

---

## 2. Cele użytkownika

System przechowuje:

- cele krótkoterminowe,
- cele średnioterminowe,
- cele długoterminowe.

Przykłady:

- poprawa kondycji,
- nauka nowych umiejętności,
- rozwój zawodowy,
- budowa relacji.

---

## 3. Zainteresowania

Informacje o:

- hobby,
- ulubionych aktywnościach,
- sposobach odpoczynku,
- rzeczach sprawiających przyjemność.

---

## 4. Nawyki

System analizuje:

- godziny aktywności,
- regularne zachowania,
- odkładane zadania,
- produktywność w różnych porach dnia.

---

## 5. Preferencje decyzyjne

System uczy się:

- jak użytkownik podejmuje decyzje,
- jakie wartości są dla niego ważne,
- jakie kompromisy akceptuje.

---

## 6. Relacje

Uwzględnia:

- rodzinę,
- przyjaciół,
- ważne osoby,
- wspólne aktywności.

---

## 7. Stan aktualny

Uwzględnia:

- aktualny poziom energii,
- obciążenie zadaniami,
- stres,
- dostępny czas.

---

# Zasada działania

Ogarniacz nie podejmuje decyzji za użytkownika.

Jego zadaniem jest:

1. zebrać kontekst,
2. przeanalizować sytuację,
3. zaproponować najlepsze rozwiązanie.

Ostateczna decyzja zawsze należy do użytkownika.

---

# Przykład

Użytkownik planuje:

"Umyć auto w środę."

System analizuje:

- prognozę pogody,
- dostępny czas,
- historię użytkownika,
- poziom obciążenia.

Sugestia:

"Środa może być nie najlepszym terminem. W czwartek rano będzie lepsza pogoda, a Twój kalendarz wygląda spokojniej. Przenieść zadanie?"
